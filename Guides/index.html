How to Use This Site

This site is designed to be your go-to reference during actual renovation work. Each guide includes:

  • Exact measurements — drill sizes, pipe diameters, wall distances, heights
  • Materials & tools lists — everything you need before you start
  • Dimensional diagrams — visual reference for layout and placement
  • Step-by-step instructions — ordered walkthroughs for each procedure
  • Pro tips & code notes — things the video may not emphasize enough
⚠️ Always Check Local Codes
Plumbing and electrical codes vary by jurisdiction (IPC vs UPC for plumbing, NEC amendments for electrical). Always verify with your local building department before starting work, and pull permits where required.
